console.log() - вывести на экран

length('Name') - выводит цифру символов этой функции( их сумма). Конкретно в этом случае выведет 4

pow(2, 3) - возведение в степень. В данном случае 2 возводим в 3 степень 2*2*2 . Ответ 8

Math - функция для математических операций
Напрмер, Math.pow(2, 3); // 8
https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Global_Objects/Math

Math.ceil() - возвращает БОЛЬШЕЕ (если дробь) или эквивалентное целое число
Math.ceil(7) = 7
Math.ceil(7.4) = 8
Math.ceil(-5.2) = -5 ( -5 БОЛЬШЕ чем -6, пэотому -5)

round() - функция, которая округляет данное число ( по умолчанию до целых). Через запятую можем подать число цифр кругления после запятой
round(10.3) = 10
round(20.34573, 2) = 20.35

Math.max([value1[, value2[, ...]]]) - максимальное число из данного набора
Math.min([value1[, value2[, ...]]]) - минимальное число из данного набора

console.log(typeof 43) - typeof показывает к какому классу принадлежит это выражение (в данном случае 43). Выведет Number

Создание и вызов функции!!!

const showGreeting = () => {
  const text = 'Hello, Hexlet!';
  console.log(text);
}

// Вызов функции
showGreeting(); // => Hello, Hexlet!

return – особая инструкция, которая берёт выражение, записанное справа, и отдаёт его наружу, тому коду, который вызвал функцию.
Как только JavaScript натыкается на return, выполнение функции на этом завершается.
Любой код после return не выполняется
